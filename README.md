# Spectral Lifter Plus

AI 生成音源などで起きやすい「高域の物足りなさ」と「シュワシュワしたノイズ感」を、macOS のネイティブアプリで整えるためのツールです。

## 今の構成
- 画面: `SwiftUI`
- 音声読み書き: `AVFoundation`
- 周波数処理: `Accelerate`
- 実行方法: `./script/build_and_run.sh`

Python や Gradio は使っておりません。  
音声処理はすべて Swift 側で完結します。

## 使い方
1. `./script/build_and_run.sh` を実行します
2. アプリで音声ファイルを選びます
3. `処理を開始` を押します
4. 入力ファイルと同じ場所に `*_lifter` 付きで保存されます

## 処理の流れ
1. 音声読み込みと 48kHz への調整
2. スペクトル解析で高域の落ち方を確認
3. スペクトルゲートでノイズ除去
4. 高域の補完とトランジェント強調
5. 帯域別のダイナミクス調整
6. 目標音量へ寄せてピークを抑制

## テスト
- `swift build`
- `swift test`

## 注意
- 音質処理はヒューリスティック中心です
- 元音源により効き方は変わります
- 長尺ファイルでは処理時間が伸びます
